# Configuration

### Environment

- `CONFIG` - alternative template context (json format).
- `VOLTGRID_PIE_DEBUG` set to `true` to enable environment variable debugging.
- `VG_CONF_PATH` - set path to `voltgrid.conf` (Default: `/usr/local/etc/voltgrid.conf`)
- `GIT_URL`
- `GIT_DST`
